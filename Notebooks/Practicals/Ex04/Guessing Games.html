A couple of hints to help you:

  • To read input from the keyboard, you can use

x = input(message)

where message is the prompt shown to the user (e.g. â€˜Make a guess:â€™ in gg.guessingGame). Whatever the user types in gets stored in the variable x. Note that x contains a string of characters, even if the user types in a number. You can use int(x) to convert the input to an integer (and float(x) if you want to convert it to a floating-point number).

  • To generate a random integer, you can call the function np.random.randint(lo,hi) where lo is the smallest integer you want to generate, and hi is one greater than the largest integer.
